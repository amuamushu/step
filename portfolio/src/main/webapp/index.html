Projects:

Snapchat Political Ads Spending vs Number of Views - Data Science Project

We looked at Snapchat Political ad spending to determine if expensive ads or inexpensive ads were targeting Spanish speakers more. To determine this, we first prepared our data by performing preprocessing tasks such as cleaning the data and Exploratory Data Analysis. Afterwards, we conducted our hypothesis test to determine that there is a discrepancy between expensive and non-expensive ads.

BTS's Rise in Popularity - Data Visualization Project

As the final project for my Introduction to Data Visualization course, I created data visualizations using HTML/CSS, Javascript, and the Highcharts API to tell the story of how BTS rose in popularity over the years. The data was collected from various sources such as Google Trends, Spotify, and Youtube Charts.

University Sentiment vs Admissions Rate - Sentiment Analysis and Webscraping Class Project

This project was done for COGS 9: Introduction to Data Science. It aims to answer a question revolving UCSD's enrollment rate: whether there is correlation betweeen media influence and the number of students enrolling every year. For this project, we defined media influence to be the sentiment score from news articles on UCSD and the school's US World News ranking.
